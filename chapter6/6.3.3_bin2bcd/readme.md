6.3.3 Binary-to-BCD conversion circuit
--------------------------------------

### Page 160

#### Testing circuit operation

 - `sw[12:0]`: binary input
 - `btnR`: start
 - `btnD`: stop
 - `led[0]`: ready
 - `led[1]`: done tick (too fast to actually see)

The BCD converted number is displayed on the 7-seg LEDs.
